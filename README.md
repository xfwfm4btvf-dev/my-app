# 🚀 Nitrogen Blog

A modern tech blog built with **Next.js 16** + **Magic UI** + **Tailwind CSS**.

## ✨ Features

- 🎨 Beautiful animated components from Magic UI
- ⚡ Next.js 16 with App Router
- 🌙 Dark mode design
- 📱 Fully responsive
- 🔍 SEO optimized

## 🔗 Live Demo

👉 **[https://xfwfm4btvf-dev.github.io/my-app/](https://xfwfm4btvf-dev.github.io/my-app/)**

## 📦 Tech Stack

- **Framework:** Next.js 16
- **UI Library:** Magic UI + Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript
- **Deployment:** GitHub Pages

## 🏃‍♂️ Getting Started

```bash
npm install
npm run dev
```
